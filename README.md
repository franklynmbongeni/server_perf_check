# server_perf_check

this is a simple python script to analyze and report server perfomance

features:
-Displays a detailed system information(OS, HostName, architacture, uptime)
-Monitors CPU usage per core
-Shows memory usage (used, available, total)
-Tracks network I/O (sent and recieved data)
-
it is easy to run for quick health checks

-REQUIREMENTS
-psutil library (to install it, PIP INSTALL PSUTIL)
